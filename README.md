# uBar 4 让 Mac 焕发新生：告别 Dock 焦虑，Windows 老兵秒上手

## 写在开头：一条任务栏，治好了我的 Mac 适应不良

我自己就是个典型的"半路出家"Mac 用户。用了十几年 Windows，突然换到 Mac，最难受的不是快捷键变了，是那条 Dock。

Dock 这东西设计挺优雅，可它就是个"应用启动器"，不是任务管理器。开十几个窗口的时候，想找一个特定的 Word 文档，得先点 Word 图标，再在窗口列表里翻半天。Windows 那种"一个窗口一个标签"的直觉，没了就是没了。

直到有天我装上了 uBar，那种熟悉感瞬间回来了。底部一条横向任务栏，每个窗口都有自己的标签，鼠标悬停还能预览——就像回到了老家，拖鞋都在原来的位置。

## 它到底能干什么：六个让我离不开的功能

**1. Windows 风格的开始菜单**

点左下角 uBar 图标，弹出开始菜单，能快速启动应用、访问文件夹、关机重启。这个对刚从 Windows 过来的人来说，简直是心理按摩。

**2. 窗口预览**

鼠标悬停在应用图标上，会浮出该应用所有窗口的实时缩略图。找文档的时候不用一个个点开看，一眼定位。

**3. 多层任务栏**

窗口太多挤不下？把鼠标移到任务栏上边缘，三指向上拖（或按住右键拖），最多能拉出 5 层空间。我开 30 个窗口的时候，这功能救过我无数次。

**4. 应用状态一目了然**

有消息的应用会闪烁提醒；按住 Control 键能看到每个应用的 CPU 和内存占用；Music、Spotify、VLC 这类播放器还能直接在任务栏显示播放进度。徽章通知是那种经典的大红点，爱不爱由你，开关在设置里。

**5. 多显示器支持**

接了外接屏的话，可以让每个屏幕显示自己的任务栏（只显示该屏幕上的窗口），也可以开启镜像模式让所有屏幕任务栏一致。创意工作者和开发者应该会喜欢。

**6. 主题自定义**

系统、明亮、黑暗、新白金、正在播放、自定义——六种主题。自定义模式下背景色、前景色、高亮色、透明度都能调，配合壁纸玩出花来。
